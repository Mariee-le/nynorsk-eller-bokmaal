# nynorsk-eller-bokmaal
Pilotprosjekt for å teste målforma til ei tekst på norsk ved å bruke nltk og korpusa Norwegian_Norsk-Nynorsk-Latin og Norwegian_Norsk-Bokmal-Latin1 frå udhr. Modellen er ikkje så veldig bra, og korpuset ikkje har varierte sjangrar. Klarer ikkje å registrer "jeg" som bokmål, eller "eg" som nynorsk. Må eventuelt legge til ord på bokmål og nynorsk manuelt.
